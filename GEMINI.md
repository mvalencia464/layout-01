# Project Overview

This is a React and TypeScript project built with Vite. It appears to be a website for a painting and contracting company. The project uses `react-router-dom` for routing, and has a large number of pages and services defined. Tailwind CSS is used for styling.

# Building and Running

*   **Development:** `npm run dev`
*   **Build:** `npm run build`
*   **Lint:** `npm run lint`
*   **Preview:** `npm run preview`
*   **Typecheck:** `npm run typecheck`

# Development Conventions

*   The project uses TypeScript and follows standard React conventions.
*   Styling is done with Tailwind CSS.
*   The project is structured with a `src` directory containing `components`, `pages`, and `content`.
*   Routing is centralized in `src/main.tsx`.
